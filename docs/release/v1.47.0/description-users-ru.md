# Helyx — управляй Claude Code из Telegram

**Helyx** — это бесплатный open-source бот, который соединяет Telegram с Claude Code, запущенным на вашем компьютере или сервере. Больше не нужно сидеть за терминалом: попросите Claude написать код, проверить PR или посмотреть статус задачи — прямо из мессенджера.

---

## Что умеет

- **Чат с Claude Code из любого места** — отправляете сообщение в Telegram, Claude Code его получает, работает с кодом и отвечает в тот же тред.
- **Голосовые сообщения** — говорите запрос голосом, Helyx транскрибирует и передаёт Claude.
- **Фото и скриншоты** — отправьте скриншот или схему, Claude увидит.
- **Кнопки вместо терминала** — когда Claude хочет отредактировать файл или запустить команду, вы получаете инлайн-кнопку «Разрешить / Отклонить» прямо в чате.
- **Живой статус** — «Думаю… (0:42)» обновляется в реальном времени.
- **Форум-топики по проектам** — каждый проект получает свой топик в форум-группе. Маршрутизация автоматическая, вручную переключаться не нужно.
- **Память** — Helyx запоминает факты о ваших проектах и использует их в следующих сессиях.
- **Голосовые ответы** — Claude может зачитать ответ через синтез речи.
- **Веб-дашборд** — браузерная панель с сессиями, памятью, логами разрешений и статистикой API.

---

## Для кого

- Разработчикам, которые хотят работать с Claude Code не сидя у терминала.
- Тем, кто хочет одобрить редактирование файла или проверить статус сборки с телефона.
- Командам, самостоятельно развёртывающим AI-ассистентов для разработки.

---

## Что нужно

Helyx работает на вашем железе — никаких внешних облаков. Потребуется:

- Компьютер или сервер с Docker и интернетом
- Установленный [Claude Code](https://claude.ai/code) с Anthropic API-ключом
- Аккаунт Telegram
- Около 15 минут на настройку

Данные остаются у вас. Helyx не отправляет ничего на сторонние сервисы, кроме сообщений через Telegram API и запросов к Anthropic API — точно так же, как при обычном использовании Claude Code.
