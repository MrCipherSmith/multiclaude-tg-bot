# ПРЕСС-РЕЛИЗ

## Helyx v1.47.0: открытый Telegram-мост для Claude Code с форум-режимом, Skills Toolkit и полной MCP-интеграцией

**1 мая 2026** — Проект Helyx выпустил версию 1.47.0 — открытый инструмент для управления сессиями Claude Code прямо из Telegram. В этом релизе: маршрутизация по форум-топикам, система переиспользуемых навыков (Skills Toolkit) и двусторонний MCP-транспорт.

Helyx позволяет разработчику держать Claude Code на сервере или ноутбуке и общаться с ним через Telegram: текст, голосовые, фото. Claude отвечает в тот же тред, показывает статус работы в реальном времени и выносит опасные действия на подтверждение — прямо в чате. Всё самостоятельно хостится, нет внешних сервисов.

### Что нового в v1.47.0

- **Режим форум-супергруппы** — каждый проект получает отдельный топик. Статус, запросы разрешений и ответы попадают в нужный топик автоматически, без ручного `/switch`.
- **Двойной MCP-транспорт** — stdio-канал на хосте доставляет сообщения в Claude Code; HTTP MCP-сервер в Docker принимает результаты вызовов инструментов обратно. Чистая двусторонняя шина.
- **Skills Toolkit** — Claude предлагает сохранить повторяемые паттерны из сессий как «навыки». Вы подтверждаете одной кнопкой. Еженедельный куратор архивирует устаревшее.
- **Гейтинг разрешений** — деструктивные действия (редактирование файлов, команды shell) требуют явного подтверждения через инлайн-кнопки в Telegram. Можно настроить авто-апрув для доверенных паттернов.
- **Семантическая память** — PostgreSQL 16 + pgvector, дедупликация через LLM (`rememberSmart`), автосуммаризация при простое сессии.
- **Живой статус** — «Думаю… (0:12)» обновляется пока Claude работает, данные берутся из tmux-панели.
- **Веб-дашборд** — React/Vite SPA на порту 3847: управление сессиями, браузер памяти, логи разрешений, статистика API, git-браузер.
- **Telegram Mini App** — лёгкая панель прямо в Telegram: git-браузер, разрешения, состояние сессий.
- **Голос** — транскрипция через Groq Whisper (или локальный фоллбэк), TTS через Piper, Kokoro, Яндекс или Groq.
- **Автономный режим** — бот может работать напрямую с Anthropic, Google AI, OpenRouter или Ollama без активной сессии Claude Code.
- **Codex-ревью** — `/codex_review` запускает OpenAI Codex CLI для ревью кода прямо из Telegram.

### Технические детали

Стек: Bun + TypeScript, Telegram-фреймворк grammY. Хранение: PostgreSQL 16 + pgvector, эмбеддинги через Ollama (`nomic-embed-text`). Дашборд: React + Vite. Деплой через Docker Compose, канальный процесс запускается на хосте под Bun. Для полноценной работы нужна установленная Claude Code с активным Anthropic API-ключом. В репозитории — готовый Docker Compose, инструменты миграций БД и шаблон `CLAUDE.md` для контекста проекта.
